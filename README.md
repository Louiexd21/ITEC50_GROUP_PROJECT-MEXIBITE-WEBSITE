Welcome! This is the official website for MexiBite, a vibrant chicken wings restaurant serving bold flavors and irresistible bites.
https://itec-50-group-project-mexibite-webs-kohl.vercel.app/

MexiBite is a modern, responsive web presence built for a local restaurant specializing in chicken wings. The site showcases the menu, promotions, location, ordering info, and brand identity — with a design that’s tasty to look at and delicious to navigate!
This project was developed as part of the ITEC-50 Group Project and aims to deliver a professional restaurant web experience.
